# Compliance Framework Reference (condensed)

## OWASP Top 10 for Agentic Applications

| ID | Description |
|----|-------------|
| ASI01 | Goal Hijacking — prompt injection redirects agent objectives |
| ASI02 | Tool Misuse — agent invokes tools in unintended or harmful ways |
| ASI03 | Identity and Privilege Abuse — agent operates beyond granted permissions |
| ASI04 | Supply Chain Vulnerabilities — compromised dependencies, plugins or MCP servers |
| ASI05 | Unsafe Code Execution — agent executes unvalidated or malicious code |
| ASI06 | Memory Poisoning — adversary corrupts agent memory or context |
| ASI07 | Inter-Agent Exploitation — one agent manipulates another via shared channels |
| ASI08 | Cascading Failures — single agent failure propagates across connected systems |
| ASI09 | Trust Boundary Violations — agent crosses trust boundaries without verification |
| ASI10 | Rogue Agents — agent operates outside human oversight or control |
